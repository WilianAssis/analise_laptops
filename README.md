# Projeto de Extensão Universitaria - Inteligência de Mercado e Analise de Dados 

##### Fonte dedados: Dataset "Laptop Price Dataset", obtido no Kaggle.

[!NOTE]
Em cenarios reais, dados como estes podem ser coletados por meio de web scraping em plataformas de e-commerce, marketplaces e comparadores de preco.
Este projeto simula uma aplicacao real de inteligencia de mercado, em que a analise de dados apoia a tomada de decisao do consumidor com base em custo-beneficio.
