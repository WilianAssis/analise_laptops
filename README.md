# Analise de Precos de Laptops

Projeto de analise de dados com foco em inteligencia de mercado e custo-beneficio na compra de notebooks.

O objetivo deste repositorio e explorar um dataset de precos de laptops para responder perguntas como:

- Quais marcas apresentam os maiores precos medios?
- Quais configuracoes tendem a oferecer melhor custo-beneficio?
- Como caracteristicas como RAM, armazenamento e processador influenciam o preco?
- Que perfis de consumidor podem ser atendidos com diferentes faixas de preco?

## Objetivo

Este projeto busca transformar dados brutos em informacoes uteis para apoiar a tomada de decisao de compra. A ideia e simular um cenario de inteligencia de mercado, em que a analise ajuda a comparar produtos e identificar padroes de valor.

## Fonte dos dados

Dataset: `Laptop Price Dataset`  
Origem: Kaggle

Em um contexto real, esse tipo de base poderia ser obtido por web scraping em e-commerces, marketplaces ou comparadores de preco.

## Estrutura do projeto

- `data/`: base de dados original e, se necessario, versoes tratadas
- `notebooks/`: exploracao, limpeza e analise dos dados
- `src/`: funcoes auxiliares para reaproveitar codigo
- `outputs/`: graficos, tabelas e resultados exportados
- `README.md`: apresentacao do projeto
- `requirements.txt`: bibliotecas utilizadas

## Etapas planejadas

- Importar e inspecionar o dataset
- Tratar valores ausentes e padronizar colunas
- Realizar analise exploratoria
- Criar visualizacoes para apoiar as conclusoes
- Identificar insights de custo-beneficio
- Organizar o repositorio para publicacao no GitHub

## Como executar

1. Criar e ativar um ambiente virtual
2. Instalar as dependencias com `pip install -r requirements.txt`
3. Abrir os notebooks para acompanhar a analise
